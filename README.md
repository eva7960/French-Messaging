# French-Messaging
A French AI chat bot built with Claude and Ollama to help people practice with their French grammar and conversational skills. Just download the index.html file and follow the setup guide!

While messenging you can click the little speaker icon next to each message to hear the audio in French. 

Below each text message is a translation button.

Clicking the gear icon in the top right allows you to customize the character you're talking to, how fluent you are in French, the formality of the conversation, the conversation topic, and the voice sound for the text messages. 

Happy learning!
